Featured Project

Predicting Students' Dropout & Academic Success

Developed a multi-class machine learning model using 4,424 student records and 37 variables to predict Dropout, Enrolled, and Graduate outcomes.

1. Exploratory Data Analysis

Identified 32% dropout rate and strong correlation between admission grades, academic performance, and final outcomes.

Target Distribution

Correlation Heatmap

2. Model Comparison

Implemented and compared five models: Regression Tree (75.59%), SVM (77.29%), KNN (59.32%), LightGBM (33.79%), and Random Forest (78.87%).

Model Accuracy Comparison

3. Final Model – Random Forest (78.87% Accuracy)

Random Forest achieved the highest performance, correctly classifying 417 graduates and 225 dropout cases. Feature importance analysis identified academic performance and financial indicators as key predictors.
